## 源码（程序）使用方法

### 一.安装Python3（如以安装请跳过，Ubuntu自带）

1.按下win键，在所有程序中找到Microsoft Store

2.在搜索框中输入Python3

3.选择安装最新的Python3版本（3.13）

或：

1.访问[Python官网下载地址](https://www.python.org/downloads/)

2.选择合适的版本进行下载

### 二.安装依赖库(未进行)

1.请先安装Releases中的vs_BuildTools.exe

2.Windows运行win_lib_install.bat,Ubuntu运行ubuntu_lib_install.bat

### 三.安装IDE

根据你的喜好安装你喜欢的IDE

之后你就可以在IDE中打开源码运行了
